Personal blog website running on Gatsby, React, and Node.js.

## Installation

Run local server:

```bash
git clone https://github.com/dev-hongwei/hongweiliu.com
cd hongweiliu.com
yarn
yarn run dev
```

Navigate to `localhost:8000`
